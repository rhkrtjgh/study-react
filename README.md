# 리액틀를 사용하는 이유

# 리액트에서 레이아웃 만들 때 사용하는 문법 3가지
JSX문법 1 : Class 넣을 땐 className
JSX문법 2 : 변수값을 사용항 때는 {} 중괄호 사용
   원래는 이렇게 사용함 document.querySelector('h4').innerHTML = post;
JSX문법 3 : style 정보 입력할 때 style={{이름 : '값'}}
